01 — About

Why I build things

My interest in technology started because I wanted to understand how things worked behind the scenes. I didn't just want to use apps and software; I wanted to learn how to create them. Over the past few years, this curiosity has developed into an interest in programming, web development, robotics, and problem-solving.

Right now, I split my time between writing clean code in Python and JavaScript, experimenting with digital logic circuits, and working on multidisciplinary projects that connect tech with practical problem-solving. I really enjoy taking an abstract idea and turning it into a working application.

Outside of my own projects, community leadership is a massive part of who I am. I love teaching what I know, working with collaborative teams under tight deadlines, and helping create spaces where more students feel welcome to explore STEM.

03 — Selected Works

🏆 2nd Place · International

Web App · Stripe Competition

RideRush

Theme Park Operations Simulator

RideRush is a data-driven web application that uses live JavaScript logic to simulate real-time theme park operations, visitor capacity, and financial metrics. I wanted to build something that required complex state management and algorithm logic. The app processes data dynamically and features an integrated CSV parser to handle mock operational stats. Earning 2nd place internationally in the Stripe Competition was an amazing milestone that pushed my understanding of software development and web programming.

MED+Search

Emergency Resource Locator

MED+Search is an application designed to help people quickly locate critical medical resources and navigate data during time-sensitive emergencies. I developed this with my team during a weekend hackathon. We wanted to create a clean, fast interface that removes confusion when people need clear data the most. Placing 3rd at Fraser Hacks taught me an immense amount about rapid prototyping, managing development lifecycles under a strict clock, and designing clean user workflows.

Physical Computing

Embedded Systems & Robotics Portfolio

Arduino & Tinkercad Circuit Collection

This is a documented collection of my robotics and hardware projects created using Arduino and Tinkercad. My layouts include multi-LED traffic light systems and environmental sensor arrays. For these projects, I focused on reading data from analog inputs (specifically A0 and A1) to trigger distinct digital outputs. It's given me a great foundation in physical computing and understanding how software interacts directly with physical hardware components.
